# ğŸ“– SCHOOL LIBRARY <a name="about-project"></a>


**SCHOOL LIBRARY** is an app that allows a user to record what books are in the library and who borrows them. The app allows you to: 
| Add new students or teachers.
| Add new books.
| Save records of who borrowed a given book and when.
